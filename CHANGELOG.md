# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2026-05-26

### Added

- **Premium Sound Library** — High-fidelity nature sounds, white noise, ASMR, and binaural beats with curated categories (Rain, Ocean, Forest, Wind, Thunder, Night, Water, Fire, Meditation, White Noise, ASMR)
- **Ambient Sound Mixer** — Multi-layer mixing with up to 6 simultaneous sounds, per-channel volume/pan/EQ controls, preset system, and crossfade support
- **Intelligent Timer System** — Countdown timer with fade-out, sleep timer, session tracking, and configurable alerts
- **3D Audio Visualizer** — Three.js-powered real-time spectrum analyzer with waveform, particles, bars, sphere, and tunnel modes
- **Vinyl UI Theme** — Retro vinyl record player interface with spinning record, tonearm interaction, and now-playing display
- **Mobile-First Architecture** — Capacitor-native iOS and Android support, responsive design, touch gestures, offline caching, background playback, and haptic feedback
- **Favorites System** — Save and organize preferred sound profiles
- **Dark/Light Theme** — Full theme support with system preference detection
- **Sound Search** — Search and filter sounds by name, category, or mood
- **Keyboard Shortcuts** — Play/pause, volume, next/previous controls
- **Accessibility** — ARIA labels, keyboard navigation, screen reader support

### Changed

- Initial release

- **Major (x.0.0)**: Breaking changes, large architectural shifts — ~6 months
- **Minor (1.x.0)**: New features, non-breaking improvements — ~4-6 weeks
- **Patch (1.0.x)**: Bug fixes, security patches, small enhancements — ~1-2 weeks
